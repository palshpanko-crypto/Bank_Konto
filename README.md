# Bank_Konto (Java)

Bank_Konto is a simple console-based banking system written in Java.
The project simulates basic banking operations such as user authentication, money transfers, and account management.

# Features

- User registration & login system
- Deposit and withdraw money
- Transfer money between accounts
- Transaction history (planned)
- Basic account management

# Architecture Concept

This project is designed as a learning example of clean code structure:

- Separation of concerns (each layer has its own responsibility)
- Business logic is separated from data storage
- Services handle all core operations
- Repositories manage data access

# Project Status

This project is still under active development and is continuously improving.

Some features will be added soon, including:

- Improved security (password handling and validation)
- Full transaction history system
- Better account separation and data modeling
- Exception-based error handling instead of simple boolean returns
- Code refactoring towards a more professional architecture
